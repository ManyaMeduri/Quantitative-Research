# Quantitative-Research - Natural Gas Price Estimator
#Overview
Built a natural gas price estimator using Python and Cubic Spline 
interpolation as part of a Quantitative Research simulation on Forage.

## What this project does
- Loads and cleans 4 years of natural gas price data
- Visualizes seasonal price trends
- Builds a Cubic Spline model to estimate prices for any date
- Extrapolates prices one year into the future

## Libraries used
- Pandas
- Matplotlib
- SciPy
- NumPy

## How to run
1. Open the .ipynb file in Google Colab or Jupyter Notebook
2. Upload the Nat_Gas__9_.csv file
3. Run all cells in order
